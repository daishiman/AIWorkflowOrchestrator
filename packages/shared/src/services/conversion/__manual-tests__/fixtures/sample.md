# サンプルMarkdownドキュメント

これはRAG Conversion Systemの手動テスト用のMarkdownファイルです。

## 概要

このドキュメントは以下の要素を含みます：

- 見出し（複数レベル）
- リンク
- コードブロック
- リスト

### 見出しレベル3

テストデータとして日本語と英語の両方を含めています。

## コードサンプル

TypeScriptの例：

```typescript
export function hello(name: string): string {
  return `Hello, ${name}!`;
}
```

Pythonの例：

```python
def hello(name):
    return f"Hello, {name}!"
```
