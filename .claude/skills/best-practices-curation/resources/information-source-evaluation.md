# 情報源評価ガイド

## 情報源の分類

### 一次情報源（最も信頼性が高い）
- 公式ドキュメント
- 学術論文
- 書籍（専門家著）
- 技術仕様書

**評価基準**:
- [ ] 著者の専門性が確認できる
- [ ] 公式または権威ある組織からの情報
- [ ] 最新の情報（発行から1年以内）

### 二次情報源（信頼性を評価して使用）
- ブログ記事
- カンファレンストーク
- チュートリアル
- Stack Overflow回答

**評価基準**:
- [ ] 著者が明確
- [ ] 実践的な内容を含む
- [ ] コミュニティで広く参照されている

### 三次情報源（検証が必要）
- SNS投稿
- 個人の経験談
- 非公式なフォーラム

**評価基準**:
- [ ] 他の情報源で裏付けがある
- [ ] 具体的な根拠が示されている

## 優先順位付け

**優先度: 高**
- 一次情報源
- 最新の情報（1年以内）
- 著者の専門性が確認できる

**優先度: 中**
- 二次情報源
- 実践的な内容
- コミュニティで参照されている

**優先度: 低**
- 三次情報源
- 古い情報（3年以上）
- 著者不明

## 採用基準

**即座に採用**:
- [ ] 一次情報源
- [ ] 品質スコア 8-10点
- [ ] 最新の情報

**補完して採用**:
- [ ] 二次情報源
- [ ] 品質スコア 6-7点
- [ ] 一部不足がある

**不採用**:
- [ ] 三次情報源のみ
- [ ] 品質スコア 0-5点
- [ ] 明らかな誤り
