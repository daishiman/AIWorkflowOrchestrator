# 品質スコアリングガイド

## 3軸評価

### 軸1: 正確性（40%の重み）

**評価基準**:

- [ ] 事実に基づいているか？
- [ ] 検証可能な根拠があるか？
- [ ] 他の信頼できる情報源と一致するか？

**スコア**:

- 8-10点: 完全に正確
- 4-7点: 概ね正確
- 0-3点: 誤りがある

### 軸2: 完全性（30%の重み）

**評価基準**:

- [ ] 必要な情報がすべて含まれているか？
- [ ] 前提条件が明示されているか？
- [ ] エッジケースが説明されているか？

**スコア**:

- 8-10点: 包括的
- 4-7点: 基本は含まれる
- 0-3点: 不完全

### 軸3: 明確性（30%の重み）

**評価基準**:

- [ ] 理解しやすいか？
- [ ] 専門用語に説明があるか？
- [ ] 例が適切か？

**スコア**:

- 8-10点: 非常に明確
- 4-7点: 理解可能
- 0-3点: 不明確

## 総合スコア計算

```
総合スコア = (正確性 × 0.4) + (完全性 × 0.3) + (明確性 × 0.3)
```

## 採用判定

- **8-10点**: 即座に採用
- **6-7点**: 補完情報を追加して採用
- **4-5点**: 再評価または情報源追加
- **0-3点**: 不採用
