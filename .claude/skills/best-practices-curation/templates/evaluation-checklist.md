# ベストプラクティス評価チェックリスト

## 情報源評価

### 一次情報源チェック

- [ ] 公式ドキュメントまたは書籍か？
- [ ] 著者の専門性が確認できるか？
- [ ] 発行から1年以内か？

### 二次情報源チェック

- [ ] 著者が明確か？
- [ ] 実践的な内容を含むか？
- [ ] 他の情報源でも言及されているか？

## 品質評価

### 正確性（40%）

- [ ] 事実に基づいているか？
- [ ] 検証可能な根拠があるか？
- [ ] 他の信頼できる情報源と一致するか？
- [ ] 誤りや矛盾がないか？

スコア: \_\_\_\_ / 10点

### 完全性（30%）

- [ ] 必要な情報がすべて含まれているか？
- [ ] 前提条件が明示されているか？
- [ ] エッジケースが説明されているか？
- [ ] 実践に必要な詳細が揃っているか？

スコア: \_\_\_\_ / 10点

### 明確性（30%）

- [ ] 理解しやすい言葉で書かれているか？
- [ ] 専門用語に説明があるか？
- [ ] 構造が論理的か？
- [ ] 例が適切に含まれているか？

スコア: \_\_\_\_ / 10点

## 総合評価

**総合スコア**: (正確性 × 0.4) + (完全性 × 0.3) + (明確性 × 0.3) = \_\_\_\_ / 10点

## 採用判定

- [ ] 8-10点: 即座に採用
- [ ] 6-7点: 補完情報を追加して採用
- [ ] 4-5点: 再評価または情報源追加が必要
- [ ] 0-3点: 不採用

## アクション

**採用の場合**:

- [ ] 統合パターンを決定（追加/強化/置換/統合）
- [ ] 関連知識との関係を明示
- [ ] バージョン履歴に記録

**不採用の場合**:

- [ ] 不採用の理由を記録
- [ ] 代替情報源を探索
