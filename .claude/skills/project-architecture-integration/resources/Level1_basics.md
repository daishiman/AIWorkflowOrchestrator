# Level 1: Basics

## 概要

プロジェクト固有のアーキテクチャ設計原則を専門とするスキル。 ハイブリッドアーキテクチャ（shared/features）、データベース設計、REST API、 テスト戦略、エラーハンドリング、CI/CDの原則をエージェント設計に統合します。

SKILL.md の内容だけで完結する基本運用を扱います。

- 対象タスクの目的と成果物を把握している
