# Level 1: Basics

## 概要

動的な機能拡張を可能にするプラグインアーキテクチャの設計を専門とするスキル。 レジストリパターン、動的ロード、依存性注入を活用し、 機能追加時の既存コード修正を不要にする拡張性の高いシステム設計を提供します。

SKILL.md の内容だけで完結する基本運用を扱います。
