# Level 2: Intermediate

## 概要

Markdown高度構文を活用した技術文書作成の専門スキル。 Mermaid図、高度なテーブル、コードブロック、数式表現等を適切に使用し、 読み手にとって分かりやすく、メンテナンス性の高いドキュメントを作成します。

resources/・scripts/・templates/ の活用を前提とした運用を整理します。

## 前提条件

- Level 1 の内容を理解している
- SKILL.md の適用範囲を説明できる

## 詳細ガイド

### 必要な知識・情報
- 主要トピック: 高度なテーブル構造 / 基本構文 / 整列の使い分け / コードブロックとシンタックスハイライト / 主要な言語識別子 / YAML Front Matterの活用
- 実務指針: 複雑なフローやシステム構造を視覚化する時 / APIやデータモデルの構造を表形式で整理する時 / コードサンプルを明確に提示する時

### 判断基準と検証観点
- 回避事項: アンチパターンや注意点を確認せずに進めることを避ける

### リソース運用
- `resources/advanced-tables.md`: カラム整列、マージセル表現、複雑なデータ構造、読みやすさ最適化（把握する知識: 高度なテーブル構造 / 基本構文 / 整列の使い分け）
- `resources/code-blocks.md`: 言語固有ハイライト、差分表示、行番号、ファイル名表示、実行可能コード（把握する知識: コードブロックとシンタックスハイライト / 基本構文 / 主要な言語識別子）
- `resources/front-matter.md`: YAMLメタデータ定義、title/version/author、ステータス管理、検索最適化（把握する知識: YAML Front Matterの活用 / 基本構文 / 必須フィールド）
- `resources/math-expressions.md`: LaTeX/KaTeX記法、インライン数式、ディスプレイ数式、数学記号（把握する知識: 数式表現（LaTeX/KaTeX） / 基本構文 / インライン数式）
- `resources/mermaid-diagrams.md`: フローチャート・シーケンス図・ER図・状態遷移図の作成ガイド、スタイリング（把握する知識: 対応図タイプ / 1. フローチャート / 2. シーケンス図）
- `resources/legacy-skill.md`: 旧SKILL.mdの全文（把握する知識: Markdown Advanced Syntax / リソース構造 / リソース読み取り）

### スクリプト運用
- `scripts/log_usage.mjs`: 使用記録・自動評価スクリプト
- `scripts/validate-mermaid.mjs`: Mermaid構文の自動検証（構文エラー、レンダリング可能性チェック）
- `scripts/validate-skill.mjs`: スキル構造検証スクリプト

### テンプレート運用
- `templates/specification-template.md`: 仕様書テンプレート（Mermaid図・高度テーブル・コードブロック統合）

### 成果物要件
- テンプレートの構成・必須項目を反映する

## 実践手順

1. 利用するリソースを選定し、適用順を決める
2. スクリプトは `--help` で引数を確認し、検証系から実行する
3. テンプレートを使い成果物の形式を統一する
4. `scripts/log_usage.mjs` で実行記録を残す
