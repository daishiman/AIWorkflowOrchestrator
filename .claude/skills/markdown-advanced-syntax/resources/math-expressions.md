# 数式表現（LaTeX/KaTeX）

## 概要

数学的な概念や式をMarkdown内で表現するためのLaTeX/KaTeX記法です。

## 基本構文

### インライン数式

```markdown
文中の数式は $E = mc^2$ のように記述します。
```

結果: 文中の数式は $E = mc^2$ のように記述します。

### ブロック数式

```markdown
$$
\sum_{i=1}^{n} x_i = x_1 + x_2 + \cdots + x_n
$$
```

## 基本的な記法

| 記法 | 結果 | 用途 |
|:-----|:-----|:-----|
| `x^2` | $x^2$ | 上付き（べき乗） |
| `x_i` | $x_i$ | 下付き（添字） |
| `\frac{a}{b}` | $\frac{a}{b}$ | 分数 |
| `\sqrt{x}` | $\sqrt{x}$ | 平方根 |
| `\sum` | $\sum$ | 総和 |
| `\prod` | $\prod$ | 総乗 |

## よく使う数式パターン

### 平均値
$$
\bar{x} = \frac{1}{n}\sum_{i=1}^{n} x_i
$$

### 標準偏差
$$
\sigma = \sqrt{\frac{1}{n}\sum_{i=1}^{n}(x_i - \bar{x})^2}
$$

### 対数
$$
\log_b(xy) = \log_b x + \log_b y
$$

## ギリシャ文字

| 記法 | 結果 | 記法 | 結果 |
|:-----|:-----|:-----|:-----|
| `\alpha` | $\alpha$ | `\beta` | $\beta$ |
| `\gamma` | $\gamma$ | `\delta` | $\delta$ |
| `\sigma` | $\sigma$ | `\mu` | $\mu$ |

## 注意事項

- すべての環境がLaTeX/KaTeXをサポートしているわけではない
- GitHubでは一部の数式がレンダリングされない場合がある
- 複雑な数式は画像として挿入することも検討
