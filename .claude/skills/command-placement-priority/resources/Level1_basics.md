# Level 1: Basics

## 概要

コマンドの配置場所と優先順位を専門とするスキル。 プロジェクトコマンド、ユーザーコマンド、MCPプロンプトの違い、 優先順位解決、名前空間の活用を提供します。

SKILL.md の内容だけで完結する基本運用を扱います。
