# 平易な言葉ガイド

## 概要

このガイドは、技術的な内容を一般ユーザーが理解できる言葉に変換するための
詳細な指針を提供します。

## 専門用語の言い換え辞典

### システム・技術用語

| 専門用語 | 平易な表現 | 使用例 |
|---------|-----------|-------|
| API | 接続方法/連携機能 | 「外部サービスとの連携機能を使用します」 |
| エンドポイント | 接続先 | 「データの接続先を設定します」 |
| リクエスト | 要求/送信 | 「情報を要求します」 |
| レスポンス | 応答/結果 | 「システムからの応答を確認します」 |
| 認証 | ログイン確認 | 「本人確認を行います」 |
| トークン | 認証情報/許可証 | 「アクセス許可証を取得します」 |
| パラメータ | 設定値/条件 | 「検索条件を入力します」 |
| クエリ | 検索条件 | 「検索条件を指定します」 |
| キャッシュ | 一時保存 | 「素早くアクセスするために一時保存されます」 |
| デバッグ | 問題の調査・修正 | 「問題を調査して修正します」 |

## 文体ガイドライン

### 主語の明確化

```markdown
❌ 曖昧:
「処理が実行されます」

✅ 明確:
「システムが処理を実行します」
または
「あなたが処理を開始します」
```

### 二重否定の回避

```markdown
❌ 二重否定:
「無効にしないでください」

✅ 肯定形:
「有効のままにしてください」
```

### 受動態から能動態への変換

```markdown
❌ 受動態:
「ファイルがアップロードされました」

✅ 能動態:
「ファイルのアップロードが完了しました」
「システムがファイルを保存しました」
```

### 名詞化の解消

```markdown
❌ 名詞化:
「データの保存の実行」

✅ 動詞形:
「データを保存する」
```

## 文の長さ最適化

### 長い文の分割パターン

**パターン1: 原因と結果の分離**

```markdown
❌ 長い文:
「ネットワーク接続が不安定な場合、データの同期に時間がかかることがあり、
その場合はしばらく待ってから再度お試しください。」

✅ 分割後:
「ネットワーク接続が不安定な場合、データの同期に時間がかかることがあります。
しばらく待ってから再度お試しください。」
```

**パターン2: 条件と手順の分離**

```markdown
❌ 長い文:
「管理者権限が必要な操作を行う場合は、設定画面から権限を確認し、
不足している場合は管理者に依頼してください。」

✅ 分割後:
「この操作には管理者権限が必要です。
1. 設定画面で権限を確認します
2. 権限がない場合は、管理者に依頼してください」
```

**パターン3: 複数の情報を項目化**

```markdown
❌ 長い文:
「保存できるファイル形式はPDF、Word、Excel、PowerPointで、
ファイルサイズは最大10MBまでです。」

✅ 分割・項目化後:
「対応ファイル形式:
- PDF
- Word
- Excel
- PowerPoint

ファイルサイズ: 最大10MB」
```

## コンテキスト別の表現

### 成功メッセージ

```markdown
✅ 良い例:
「保存しました！変更内容は自動的に反映されます。」
「アカウントを作成しました。確認メールをお送りしました。」
```

### エラーメッセージ

```markdown
✅ 良い例:
「メールアドレスの形式が正しくありません。
例: example@domain.com のように入力してください。」

「ファイルが大きすぎます（25MB）。
10MB以下のファイルを選択してください。」
```

### 確認メッセージ

```markdown
✅ 良い例:
「この操作を行うと、データが完全に削除されます。
削除したデータは復元できません。続けますか？」
```

## 数値と単位の表現

### 時間

| 避ける | 推奨 |
|-------|------|
| 120秒 | 2分 |
| 3600秒 | 1時間 |
| 86400秒 | 1日 |

### ファイルサイズ

| 避ける | 推奨 |
|-------|------|
| 10485760バイト | 10MB |
| 1073741824バイト | 1GB |

### 表現の統一

```markdown
✅ 一貫した表現:
「処理には約5分かかります」
「最大10MBまでのファイルをアップロードできます」
「毎日午前3時に自動バックアップが実行されます」
```
