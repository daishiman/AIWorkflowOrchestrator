### 使用例

```markdown
# タスク
以下のコードの問題を特定し、修正してください。

# 分析手順
1. コードの目的を理解する
2. 各行の処理を追跡する
3. 問題箇所を特定する
4. 修正案を提示する
5. 修正の正しさを検証する

# コード
```python
def calculate_average(numbers):
    total = 0
    for num in numbers:
        total += num
    return total / len(numbers)
```

# 期待する動作
数値リストの平均を計算する

# 実際の動作
空リストでZeroDivisionErrorが発生

# 分析

## 1. コードの目的
リスト内の数値の平均値を計算する関数

## 2. 処理の追跡
- total = 0 で合計を初期化
- forループで各数値を加算
- len(numbers)で割って平均を計算

## 3. 問題箇所
- numbersが空リストの場合、len(numbers) = 0
- 0での除算でZeroDivisionErrorが発生

## 4. 修正案
```python
def calculate_average(numbers):
    if not numbers:
        return 0  # または None、または例外を投げる
    total = 0
    for num in numbers:
        total += num
    return total / len(numbers)
```

## 5. 検証
- 空リスト: 0を返す（エラーなし）
- [1, 2, 3]: (1+2+3)/3 = 2.0
- [10]: 10/1 = 10.0
```
