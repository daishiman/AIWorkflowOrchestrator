# Level 1: Basics

## 概要

GitHub Actionsのアーティファクト管理スキル。 ビルド成果物のアップロード・ダウンロード、ジョブ間/ワークフロー間でのデータ共有、 保持期間設定、パス指定パターン、クリーンアップ戦略を提供。

SKILL.md の内容だけで完結する基本運用を扱います。

## 前提条件

- SKILL.md の概要とワークフローを読了している
- 対象タスクの目的と成果物を把握している
