# Level 1: Basics

## 概要

エージェントテンプレートと設計パターンを専門とするスキル。 4タイプのエージェントテンプレート（分析、実装、オーケストレーター、デプロイ）、 {{variable}}形式による抽象化、抽象度バランス、概念要素設計の原則を提供。

SKILL.md の内容だけで完結する基本運用を扱います。
