# Level 1: Basics

## 概要

エージェント依存関係とインターフェース設計を専門とするスキル。 スキル参照、コマンド連携、エージェント間協調のプロトコルを定義し、 循環依存を防ぎながら効果的なマルチエージェントシステムを構築します。
