# Level 1: Basics

## 概要

大容量ファイルのマルチパートアップロードを専門とするスキル。 アンドリュー・タネンバウムの『コンピュータネットワーク』に基づき、 ネットワークの不安定性を前提とした堅牢なファイル転送を設計します。

SKILL.md の内容だけで完結する基本運用を扱います。

- 対象タスクの目的と成果物を把握している
