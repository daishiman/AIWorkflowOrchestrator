# 基本Few-Shotテンプレート

## 概要

汎用的なFew-Shotプロンプトの基本テンプレート集。
各テンプレートをカスタマイズして使用してください。

## テンプレート1: 分類タスク

```markdown
# タスク説明

[入力を指定されたカテゴリに分類してください]

# カテゴリ定義（必要に応じて）

- カテゴリA: [定義]
- カテゴリB: [定義]
- カテゴリC: [定義]

# 例

例1:
入力: [入力例1]
カテゴリ: [カテゴリ]

例2:
入力: [入力例2]
カテゴリ: [カテゴリ]

例3:
入力: [入力例3]
カテゴリ: [カテゴリ]

# 実際のタスク

入力: {{input}}
カテゴリ:
```

### 使用例

```markdown
# タスク説明

メールを「緊急」「通常」「低優先」に分類してください。

# カテゴリ定義

- 緊急: 当日対応が必要
- 通常: 2-3営業日以内に対応
- 低優先: 時間がある時に対応

# 例

例1:
入力: 本日中に承認が必要です。至急ご確認ください。
カテゴリ: 緊急

例2:
入力: 今週中にレビューをお願いします。
カテゴリ: 通常

例3:
入力: 参考までに共有します。お時間のある時にご覧ください。
カテゴリ: 低優先

# 実際のタスク

入力: 明日の会議資料です。事前にご確認いただければ幸いです。
カテゴリ:
```

## テンプレート2: データ抽出

```markdown
# タスク説明

[テキストから指定された情報を抽出してください]

# 出力形式

{
"field1": "[説明]",
"field2": "[説明]",
"field3": "[説明]"
}

# 注意事項

- 情報がない場合は null を使用
- [その他の注意事項]

# 例

例1:
入力: [入力例1]
出力:
{
"field1": "[値]",
"field2": "[値]",
"field3": "[値]"
}

例2:
入力: [入力例2 - 一部欠損]
出力:
{
"field1": "[値]",
"field2": null,
"field3": "[値]"
}

# 実際のタスク

入力: {{input}}
出力:
```

### 使用例

```markdown
# タスク説明

イベント情報を抽出してください。

# 出力形式

{
"event_name": "イベント名",
"date": "日付（YYYY-MM-DD）",
"location": "開催場所",
"price": "参加費（数値、円単位）"
}

# 注意事項

- 情報がない場合は null を使用
- 日付は ISO 8601 形式で出力

# 例

例1:
入力: 「AI勉強会を12月15日に東京駅近くの会議室で開催します。参加費は3000円です。」
出力:
{
"event_name": "AI勉強会",
"date": "2024-12-15",
"location": "東京駅近くの会議室",
"price": 3000
}

例2:
入力: 「来月のネットワーキングイベント、参加無料です！」
出力:
{
"event_name": "ネットワーキングイベント",
"date": null,
"location": null,
"price": 0
}

# 実際のタスク

入力: Python入門セミナーを1月20日にオンラインで開催。参加費5000円。
出力:
```

## テンプレート3: 変換タスク

```markdown
# タスク説明

[入力を指定された形式/スタイルに変換してください]

# 変換ルール

1. [ルール1]
2. [ルール2]
3. [ルール3]

# 例

例1:
変換前: [入力例1]
変換後: [出力例1]

例2:
変換前: [入力例2]
変換後: [出力例2]

例3:
変換前: [入力例3]
変換後: [出力例3]

# 実際のタスク

変換前: {{input}}
変換後:
```

### 使用例

```markdown
# タスク説明

カジュアルな表現をビジネス敬語に変換してください。

# 変換ルール

1. 丁寧語・尊敬語・謙譲語を適切に使用
2. 口語表現を文語表現に
3. 簡潔さを保ちながら丁寧に

# 例

例1:
変換前: ありがとう！
変換後: 誠にありがとうございます。

例2:
変換前: それ、違うと思う
変換後: 恐れ入りますが、認識が異なるように存じます。

例3:
変換前: 資料送ったから見といて
変換後: 資料をお送りいたしましたので、ご確認いただけますと幸いです。

# 実際のタスク

変換前: これ、確認してくれる？
変換後:
```

## カスタマイズガイド

### プレースホルダー

| 記号        | 説明                     |
| ----------- | ------------------------ |
| `{{input}}` | 実際の入力を挿入         |
| `[説明]`    | カスタマイズが必要な箇所 |

### テンプレート選択基準

| タスクタイプ     | 推奨テンプレート |
| ---------------- | ---------------- |
| カテゴリ分類     | テンプレート1    |
| 情報抽出         | テンプレート2    |
| フォーマット変換 | テンプレート3    |
| コンテンツ生成   | テンプレート4    |
| テキスト要約     | テンプレート5    |
