## 非機能要件（該当する場合）

### パフォーマンス

```gherkin
Scenario: 応答時間
  Given [通常の負荷条件]
  When [アクション]を実行する
  Then 応答時間は[X]ms以内である
```

### セキュリティ

```gherkin
Scenario: 権限チェック
  Given [権限のないユーザー]でログインしている
  When [保護されたアクション]を実行する
  Then アクセス拒否される
    And エラーメッセージ「[メッセージ]」が表示される
```

### アクセシビリティ

```gherkin
Scenario: キーボード操作
  Given [画面]が表示されている
  When キーボードのみで操作する
  Then すべての機能が利用可能である
```
