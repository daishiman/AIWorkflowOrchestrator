# Level 1: Basics

## 概要

Drizzle ORMを活用したデータベース操作のベストプラクティスを提供するスキル。 型安全なスキーマ定義、クエリビルダーの効果的な使用、 パフォーマンスを考慮した実装パターンを提供します。

SKILL.md の内容だけで完結する基本運用を扱います。
